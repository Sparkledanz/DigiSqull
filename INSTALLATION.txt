DigiSqull Design System - Installation & Setup Guide
====================================================

REQUIREMENTS
- Node.js 18+ (check with: node --version)
- npm or pnpm (check with: npm --version)
- A modern web browser

QUICK START (1 minute)
1. npm install
2. npm run dev
3. Open http://localhost:3000

WHAT YOU'LL SEE
- Live component showcase
- Color palette display
- Typography scale
- Spacing system
- Course cards
- Video marketplace
- Transport routes
- Leaderboards
- Achievement badges
- Light/dark theme toggle (top right)

STORYBOOK (Component Browser)
1. npm run storybook
2. Open http://localhost:6006
3. Browse components in left sidebar
4. Try interactive examples
5. View component code and documentation

FILE STRUCTURE
app/
  ├── page.tsx              ← Demo showcase
  ├── layout.tsx            ← Theme provider
  └── globals.css           ← Design tokens

components/
  ├── layouts/              ← Header, PageShell, Grid
  ├── patterns/             ← CourseCard, VideoCard, etc.
  └── ui/                   ← Primitive components (50+)

lib/
  ├── design-tokens.ts      ← Token definitions
  ├── theme-context.tsx     ← Theme provider hook
  └── rn-theme-mapping.ts   ← React Native themes

.storybook/                 ← Component documentation
